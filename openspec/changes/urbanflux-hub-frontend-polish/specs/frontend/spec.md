# Delta for Frontend

## ADDED Requirements

### Requirement: Full-Width Top Bar
The top navigation bar MUST span the entire width of the screen, sitting above the sidebar and map content.

### Requirement: Top Bar Utility Information
The top bar MUST display aesthetic utility information, such as the current live time, to make it look less empty.

### Requirement: Visible Sidebar Labels
The Sidebar MUST permanently display text labels next to its icons.

### Requirement: Map Legends
A map legend MUST be rendered over the map detailing the color coding of the various transportation modes.

### Requirement: Interactive Map Node Details
Clicking a transportation node on the map MUST display a detail card (popup) showing the corridor code, departure time, and arrival time.

### Requirement: Mode Color Differentiation
Transportation modes MUST be color-coded on the map:
- Bus: Green
- Train: Yellow
- MRT: Blue
- Tram: Pink

## MODIFIED Requirements

### Requirement: Top Bar Glassmorphism
The top bar MUST use a heavy glassmorphism transparent effect (e.g., 40% opacity with strong background blur) rather than an opaque or nearly-invisible styling.
